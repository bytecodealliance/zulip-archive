Topic: custom-page-size support has landed in `wasm-ld`

fitzgen (he/him) (Mar 04 2025 at 18:05):

https://github.com/llvm/llvm-project/pull/128942

  • You can specify a custom page size via --page-size=N (where N must currently be 1 or 65536)

  • You can write page-size-agnostic code by using the __wasm_first_page_end symbol, whose address is the byte after the first page, a.k.a the first byte of the second page, a.k.a the wasm page size.

This isn't in an LLVM release yet, so you'll have to build wasm-ld yourself to test it out currently, but we're looking into backporting it to the 20.X release branch.
