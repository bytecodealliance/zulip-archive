Topic: Maintainers for wasm32-wasip1 Rust target

Alex Crichton (Jul 07 2025 at 17:31):

Currently there's only one listed maintainer for the wasm32-wasip1 Rust target (me) but the target is also considered "Tier 2" in the Rust project. Technically, though, Rust requires 2+ maintainers for a "Tier 2" standing and this target was grandfathered in from the ages of old.

The Rust project is looking to start enforcing the minimum number of maintainers, however. There's no plans to remove Tier 2 targets just yet and the MCP explains how this is the "let's get the ball rolling" part as opposed to "the axe is currently falling". Nevertheless I wanted to raise awareness here.

If anyone else would like to be listed as a target maintainer please feel free to volunteer! Rust's target tier policy is here and AFAIK this more-or-less boils down to "please be available for questions from Rust maintainers and occasionally fixing breakages if things become urgent". I at least personally don't spend too too much time maintaining these targets myself insofar that it's not a large time sink.

If you're interested in becoming a maintainer I believe the process is to add your name here.
