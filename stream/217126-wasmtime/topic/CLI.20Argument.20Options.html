Alex Crichton (Dec 05 2025 at 15:13):

The long form of -S is --wasi, and yes all those options are related to WASI configuration. Options are, in general, free form and cover all sorts of functionality that we need to cover. This includes gated/unstable proposals but it also includes configuration like enabling/disabling the network/etc.

The -S flag is what Wasmtime supports, but if you'd like more granular or different permissions please feel free to file an issue! Implementation shouldn't be too hard for things like that
