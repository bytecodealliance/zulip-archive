ComponentizeJS v0.18.4 has been released :tada:

https://www.npmjs.com/package/@bytecodealliance/componentize-js/v/0.18.4
https://github.com/bytecodealliance/ComponentizeJS/releases/tag/0.18.4

There a few important updates in this release, in particular:

  • Implement EventTarget and Event builtin (#220)
  • Add support for two-argument has and delete in URLSearchParams (#236)
  • Update of StarlingMonkey to commit 1f6f81f
  • Fix of some feature enable/disable options related to HTTP

Please download the new version and give it a try.
