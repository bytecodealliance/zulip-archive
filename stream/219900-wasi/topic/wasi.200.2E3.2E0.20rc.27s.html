We now have automation for creating WASI 0.3.0 release rc's :tada: There's more to automate and there will certainly be more release candidates, but here's to getting started on stabilizing WASI P3 API's!!!

The git tag in the wasi repos is v0.3.0-rc-2025-08-15 as well as OCI artifacts tagged with 0.3.0-rc-2025-08-15.

wasm-tools component wit webassembly_wasi_http.wasm | less

Special thanks to @Lann Martin and @Calvin Prewitt for jumping on a wkg issue that let this rollout happen today :heart: Also @Yosh Wuyts for being an awesome pair while working on the automation!
