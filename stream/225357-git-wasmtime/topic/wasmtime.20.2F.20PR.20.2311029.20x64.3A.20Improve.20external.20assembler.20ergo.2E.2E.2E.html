Especially in the case of pseudo-instructions in x64/inst/emit.rs the goal is to reduce the amount of boilerplate necessary to emit instructions. This is done via:

  • A new helper trait was added that's implemented for all assembler instructions to avoid a .into() after constructors as well as to avoid Inst::External { inst } construction.

  • Pseudo-instructions are moved to using Gpr and friends instead of Reg to more naturally convert to raw assembler types.

  • Some usage of Inst::* helpers is removed when it's known exactly what assembler instruction is going to be used and it's easier to use the assembler instruction instead.
