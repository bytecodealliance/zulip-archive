arithmetic.isle

  • ((x - y) - (x - z)) --> (z - y)
  • min(x, y) + max(x, y) --> x + y

  • (x - (x + y)) --> -y
  • (x + (y + (z - x))) --> (y + z)
  • (x + y) == (y + x) --> true
  • (x - y) != x --> y != 0

bitops.isle

  • ((x & ~y) - (x & y)) --> ((x ^ y) - y)
  • (x & ~y) | (~x & y) --> (x ^ y)
  • (x & ~y) | (x ^ y) --> (x ^ y)
  • (x & ~y) ^ ~x --> ~(x & y)
