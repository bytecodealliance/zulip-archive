Split GC generator into modules (no new features)

This PR splits the GC generator into separate modules to keep it modular and easy to maintain. No new features have been added.

  • ops.rs: Generates/introduces ops and handles encoding
  • limits.rs: Defines limits for the number of opcodes
  • mutator.rs: Implements mutations for the fuzzer
  • types.rs: Stack types, subtypes, and their implementations
  • tests.rs: Tests moved to a dedicated file

Rename: table_ops has been renamed to gc_ops.
